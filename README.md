# 📱 تطبيق إدارة المشاريع | Project App

تطبيق ويب بسيط ومنظم يهدف لمساعدة المستخدمين في تتبع مشاريعهم وإدارة مهامهم بشكل فعال.

## 🌟 المميزات (Features)

- **واجهة بسيطة:** تصميم يركز على سهولة الاستخدام والوضوح.
- **سرعة الأداء:** الموقع خفيف جداً ويحمل بسرعة فائقة.
- **متوافق مع الموبايل:** يعمل بكفاءة على جميع مقاسات الشاشات.

## 🛠️ التقنيات المستخدمة (Tech Stack)

- **HTML5:** لبناء العناصر والمدخلات.
- **CSS3:** لتنسيق الواجهة واستخدام الألوان العصرية.
- **JavaScript:** لإضافة الوظائف التفاعلية داخل التطبيق.
