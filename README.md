# 🚑 غيث لخدمات الطوارئ | Ghaith Emergency

تطبيق ويب خدمي يجمع كل أرقام النجدة والطوارئ في مكان واحد، ليسهل على المستخدم الوصول للخدمة المطلوبة بسرعة في اللحظات الحرجة.

## 🌟 المميزات (Features)
* **سرعة الوصول:** أرقام الإسعاف والشرطة والمطافي بضغطة زر واحدة.
* **تصميم واضح:** استخدام ألوان تعبر عن الحالة (الأحمر للطوارئ) مع سهولة القراءة.
* **خدمات شاملة:** يضم نجدة الطفل، طوارئ الغاز، والكهرباء بجانب الخدمات الأساسية.

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **HTML5:** لبناء الهيكل وتوزيع البطاقات (Cards).
* **CSS3:** لتصميم الواجهة العصرية المتجاوبة.
* **JavaScript:** لجعل أزرار "اتصل الآن" تعمل بكفاءة.
